# Web3 Archive

A curated registry of Web3 project launches.

## About

Web3 Archive is a simple, beautiful way to explore when major cryptocurrency and blockchain projects launched. Browse through a calendar view, timeline, or curated list to discover the history of Web3.

## Features

- **Calendar View** - See project launches on a monthly calendar grid
- **Timeline** - A chronological stream of all projects
- **Curated List** - A clean table view with project details

Each project includes launch date, blockchain network, token symbol, category, and links to official resources.

## Built With

Built by Google Antigravity using Next.js, React, Tailwind CSS, and PostgreSQL.
